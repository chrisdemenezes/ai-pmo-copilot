# AI Agent Architecture

## Agent Model

The platform uses specialized AI agents coordinated by an orchestration layer.

## Core Agents

### Project Status Agent
Responsibilities:
- Analyze project health
- Identify deviations
- Generate executive summaries

### Meeting Intelligence Agent
Responsibilities:
- Process transcripts
- Extract decisions
- Create action items

### Risk Advisor Agent
Responsibilities:
- Analyze risks
- Predict impacts
- Recommend mitigations

## Agent Flow

Input Data -> Agent Processing -> Validation -> Business Output

## Human in the Loop

Critical recommendations require user validation before execution.
