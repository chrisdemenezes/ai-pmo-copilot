# Agent Specifications

## Project Status Agent

Input:
- Project plans
- Status reports
- Issues
- Risks

Output:
- Health assessment
- Key findings
- Recommendations

## Meeting Intelligence Agent

Input:
- Meeting transcripts
- Notes

Output:
- Minutes
- Decisions
- Action items

## Risk Advisor Agent

Input:
- Project indicators
- Historical data

Output:
- Risk matrix
- Mitigation actions
