# User Stories

## Status Analysis
As a Project Manager, I want AI to analyze project status so that I can identify problems earlier.

## Meeting Summary
As a PMO Analyst, I want automatic meeting summaries so that governance information is captured consistently.

## Risk Management
As an IT Manager, I want AI recommendations on risks so that decisions are faster and more accurate.

## Executive Reporting
As an Executive Sponsor, I want concise dashboards so that I can understand portfolio health.
