Princípios permanentes consolidados nesta Capability

PrincípioDefinição
Progressive ContextA STRATECH não navega entre telas, ela aprofunda contexto — cada Capability entrega um nível maior de compreensão do mesmo objeto (Dashboard → Portfolio Intelligence → Workspace → Executive Brief)
Progressive PurposeCada nível da plataforma tem um propósito próprio — reaproveitar um mesmo sinal entre duas telas não é duplicação quando a pergunta executiva de cada uma difere
Executive Responsibility SeparationDuas Capabilities podem usar os mesmos sinais reais, mas nunca devem responder à mesma pergunta executiva — a responsabilidade funcional de cada uma permanece distinta
Executive Navigation (framework conceitual)Sequência natural: Onde devo olhar? → Portfolio Intelligence → O que depende de mim? → Decision Center → O que preciso entender? → Workspace → O que significa? → Executive Brief → O que preciso fazer? → Action Intelligence.
