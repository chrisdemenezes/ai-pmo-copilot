STRATECH V1 RC-1 — como rodar localmente

1. Clonar

git clone <url-do-repositorio> ai-pmo-copilot
cd ai-pmo-copilot

2. Configuração

Nenhuma configuração manual é necessária. A primeira execução cria automaticamente demo/.env com uma senha de sessão gerada e o Demo Mode ativado (sem chave de IA real necessária).

3. Setup (uma vez por máquina)

macOS/Linux — o próprio start já inclui o setup:

bash scripts/rc1-local-start.sh

Windows — rodar uma vez:

setup.bat

4. Start

macOS/Linux — o comando do passo 3 já inicia tudo. Windows:

start.bat

Aguardar ~15-30s até os dois serviços responderem.

5. URL

→http://localhost:3000/entrar

6. Login Demo

→Senha do workspace: demo-local-password (confirmar em demo/.env se personalizada)

Para popular o portfólio fictício (primeira vez): python3 demo/seed_demo_data.py

7. Como parar

macOS/Linux: bash demo/stop-demo.sh. Windows: stop.bat.
